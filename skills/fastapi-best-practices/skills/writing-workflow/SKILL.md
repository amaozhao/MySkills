# Writing Workflow

全流程文章写作助手，从选题到复盘。

## 工作流程

```
选题 → 选题确认（硬门槛）→ 大纲 → 大纲确认（硬门槛）→ 写作 → 标题 → 发布
```

> **重要**：每个阶段有"硬门槛"，必须用户确认后才能进入下一阶段

## 快速指令

| 指令 | 进入阶段 |
|------|----------|
| `帮我选题` | 选题阶段 |
| `帮我写大纲` | 大纲阶段 |
| `帮我写作` | 写作阶段 |
| `帮我起标题` | 标题阶段 |
| `准备发布` | 发布阶段 |
| `分析数据` | 复盘阶段 |

## 支持平台

| 平台 | 特点 | 建议字数 |
|------|------|----------|
| 公众号 | 长文、深度 | 2000-3000字 |
| 知乎 | 逻辑、专业 | 1500-2500字 |
| 小红书 | 短平快、颜值 | 500-1000字 |
| 博客 | 灵活 | 可长可短 |

## 使用方式

1. **直接说需求**：`帮我写一篇关于 AI 工具对比的文章`
2. **指定平台**：`帮我写一篇公众号文章，选题是...`
3. **指定阶段**：`帮我起标题` → 进入标题阶段
