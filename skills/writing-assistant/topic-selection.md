# 选题助手

## 概述

帮助你确定写什么主题，包含自动搜索热门话题、选题确认和决策流程。

## 工作流程

```
选题灵感 → 热门搜索 → 选题分析 → 选题确认（硬门槛）→ 5问决策 → 确定选题
```

## 阶段一：自动搜索热门话题

当用户需要选题时，自动执行：

```python
# 搜索当前热门话题
搜索 "2026年 AI工具 程序员 热门话题"
```

### 搜索结果示例

```
【热门话题搜索】
根据你关注的领域「AI工具」，搜索到以下热门话题：

1. Trae 中国版发布一周体验
   - 来源：技术社区
   - 热度评估：⭐⭐⭐⭐
   - 时效性：最佳（7天内）

2. Claude Code 插件开发
   - 来源：开发者社区
   - 热度评估：⭐⭐⭐
   - 时效性：较佳（14天）

3. AI 编程助手对比
   - 来源：社交媒体
   - 热度评估：⭐⭐⭐⭐
   - 时效性：长期热门

【请选择】：请告诉我你感兴趣的选题编号
```

---

## 阶段二：选题确认（硬门槛）

借鉴 copywriting 的 "Copy Brief Lock"，在进入大纲阶段前，必须确认选题：

```
【选题确认】
请确认以下选题信息：

1. 选题：[你选择的选题]
2. 目标读者：[谁会看这篇文章]
3. 核心价值：[读者能获得什么]
4. 独特角度：[你的差异化是什么]

✅ 请回复"确认"进入大纲阶段
```

### 硬门槛规则

- ❌ 未明确目标读者 → 不进入下一阶段
- ❌ 未说明独特价值 → 不进入下一阶段
- ✅ 用户确认后 → 正式进入大纲阶段

---

## 5 问详细说明

| 问题 | 判断标准 | 是 | 否 |
|------|---------|-----|-----|
| 1. 目标读者会关心吗？ | 选题与读者相关 | 继续 | 放弃 |
| 2. 时效性还强吗？ | 48小时内/7天内/仍有时效 | 继续 | 放弃/延期 |
| 3. 有独特角度吗？ | 差异化价值 | 继续 | 放弃 |
| 4. 你能提供独特价值吗？ | 专业经验/亲身经历 | 继续 | 放弃 |
| 5. 有时间高质量完成吗？ | 时间充裕 | 开始写 | 延期 |
