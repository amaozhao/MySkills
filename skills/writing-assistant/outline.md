# 大纲助手

## 概述

根据选题和文章类型生成大纲模板。

## 工作流程

```
选题确认 → 选择文章类型 → 生成大纲 → 大纲确认（硬门槛） → 进入写作
```

## 阶段一：选择文章类型

请用户选择文章类型：

```
【选择文章类型】
1. 工具对比 - 比较两个或多个工具
2. 情绪共鸣 - 讲述个人经历，引发共鸣
3. 热点解读 - 解读当前热点事件
4. 日常感悟 - 分享生活感悟

请回复编号或描述你的类型
```

---

## 阶段二：大纲确认（硬门槛）

借鉴 copywriting 的 "Copy Brief Lock"，生成大纲后必须确认：

```
【大纲确认】

选题：Trae 中国版发布一周体验
类型：工具对比
目标读者：程序员/AI 工具使用者

大纲结构：
01. 发生了什么（200字）
02. 为什么值得关注（500字）
03. 核心差异点1：免费vs付费（400字）
04. 核心差异点2：本地模型（400字）
05. 我的推荐（300字）
06. 写在最后（200字）

预计字数：2000字
平台：公众号

✅ 请回复"确认大纲"进入写作阶段
```

### 硬门槛规则

- ❌ 大纲未完整 → 不进入写作
- ❌ 字数不明确 → 不进入写作
- ✅ 用户确认后 → 正式进入写作阶段

## 4 种文章类型

### 类型一：工具对比

**适用场景**：比较两个或多个工具、产品的优劣。

**读者价值**：帮助读者做出选择决策。

**大纲模板**：

```
【标题】使用XX和XX后，我发现了X个关键差异

【开头（150字以内）】
- 场景代入：什么场景下使用的
- 发现：有什么问题或重要发现
- 预告：本文会告诉你什么

【正文结构】
一、先说结论（300字）
- 两个工具的定位和特点
- 我的推荐（针对不同需求）

二、核心对比（800字）
- 功能对比
- 性能对比
- 性价比对比
- 每个对比点用实际案例说明

三、适用场景（500字）
- 什么情况用A更好
- 什么情况用B更好
- 什么情况可以都用

四、最终建议（300字）
- 明确推荐
- 给出使用建议

【结尾】
- 总结核心观点
- 引导互动讨论
```

**字数参考**：2000-3000字

---

### 类型二：情绪共鸣

**适用场景**：讲述个人经历，引发读者情感共鸣。

**读者价值**：获得情感支持、行动启发。

**大纲模板**：

```
【标题】关于XX的那段经历，我有些话想说

【开头（200字以内）】
- 场景：具体的时间、地点、情境
- 情绪：那一刻的真实感受
- 转折：我决定如何面对

【正文结构】
一、那个瞬间（400字）
- 详细描述发生了什么
- 细节越多越好（增强代入感）
- 真实呈现，不刻意渲染

二、为什么难（500字）
- 不是简单的困难，是深层的挑战
- 触及真正的痛点
- 引发读者共情

三、我做了什么（600字）
- 经历了哪些挣扎和尝试
- 如何一步步走出来的
- 关键转折点和突破

四、给同样处境的你（500字）
- 不是空洞的鸡汤，是具体的建议
- 可操作性强
- 真诚不夸大

【结尾】
- 一句话总结核心感悟
- 引导互动交流
```

**字数参考**：1800-2500字

---

### 类型三：热点解读

**适用场景**：解读当前热点事件，提供独特视角。

**读者价值**：了解事件背后的逻辑、获得独特观点。

**大纲模板**：

```
【标题】关于XX，我来分享一些不同的看法

【开头（150字以内）】
- 简单交代发生了什么
- 你的独特立场或角度

【正文结构】
一、发生了什么（300字）
- 客观描述事件经过
- 关键时间节点

二、为什么引发关注（400字）
- 分析传播原因
- 不同角度看问题

三、深度解读（800字）
- 你的独特分析角度
- 从多个层面剖析
- 不是简单复述，是深度思考

四、对目标读者的意义（500字）
- 利好哪些人
- 需要注意什么
- 有什么启示

五、我的判断和建议（300字）
- 趋势预测
- 行动建议

【结尾】
- 核心观点总结
- 引导讨论交流
```

**字数参考**：2000-3000字

---

### 类型四：日常感悟

**适用场景**：分享日常生活感悟、心得。

**读者价值**：获得启发、情感共鸣。

**大纲模板**：

```
【标题】关于XX，我的一点思考

【开头（150字以内）】
- 引入话题
- 抛出核心观点

【正文结构】
一、背景交代（300字）
- 发生了什么
- 我的处境和情境

二、核心观点（600字）
- 我的深入思考
- 我的实践和体验
- 具体的收获和感悟

三、经验总结（400字）
- 我学到了什么
- 对你可能有何启发

【结尾】
- 升华主题
- 引导互动交流
```

**字数参考**：1500-2000字
