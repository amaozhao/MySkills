# 发布助手

## 概述

帮助你准备文章发布，包括平台配置、发布时间建议、发布前检查。

## 工作流程

```
标题确认 → 平台配置 → 发布时间 → 发布前检查 → 发布
```

## 输出规范

借鉴 seo-content-writer，每次发布输出：

```
【发布检查清单】

✅ 平台配置
- 公众号：[封面图、摘要、标签]
- 知乎：[话题、专栏]
- 小红书：[话题标签]

✅ 发布时间
- 推荐时间：[具体时间]
- 理由：[为什么这个时间最好]

✅ 内容检查
- [ ] 开头100字有钩子
- [ ] 有真实案例
- [ ] 结尾有引导

✅ 标题检查
- [ ] 标题符合平台规范

✅ 发布检查
- [ ] 平台账号登录
- [ ] 定时发布已设置
```

## 平台配置

### 微信公众号

| 配置项 | 说明 |
|--------|------|
| 封面图 | 16:9 封面图 |
| 摘要 | 40字以内 |
| 标签 | 相关领域标签 |
| 原创声明 | 如需声明 |

### 知乎

| 配置项 | 说明 |
|--------|------|
| 标题 | 可与公众号不同 |
| 封面 | 竖图为主 |
| 话题 | 添加相关话题 |
| 专栏 | 同步到专栏 |

### 小红书

| 配置项 | 说明 |
|--------|------|
| 封面 | 竖图 4:3 |
| 标题 | 20字以内 |
| 正文 | 可用 emoji |
| 话题 | #话题# 格式 |

### 博客

| 配置项 | 说明 |
|--------|------|
| 标签 | SEO 关键词 |
| 分类 | 文章归类 |
| 摘要 | SEO 描述 |

## 发布时间

### 黄金时间段

| 时间段 | 特点 | 适用场景 |
|--------|------|----------|
| 早 07:00-08:30 | 通勤，碎片刷 | 短内容、快阅读 |
| 午 12:00-13:30 | 午休，躺床上刷 | 中等内容、情感类 |
| 晚 20:00-22:00 | 黄金时间，注意力集中 | 深度内容、干货类 |

### 按内容类型

| 内容类型 | 推荐时间 | 理由 |
|----------|----------|------|
| 深度长文 | 晚 20:00-21:00 | 用户有耐心读完 |
| 追热点 | 越快越好 | 时效性优先 |
| 情感共鸣 | 晚 21:00 左右 | 互动率高 |
| 实用工具 | 早 07:30-08:00 | 通勤时查看 |

## 发布前检查清单

### 内容检查

- [ ] 开头 100 字有钩子
- [ ] 有至少 1 个真实故事或案例
- [ ] 有实质性内容或观点
- [ ] 结尾有总结和引导

### 标题检查

- [ ] 标题能引起目标读者兴趣
- [ ] 标题清晰表达文章价值
- [ ] 标题无歧义

### 排版检查

- [ ] 有合适的配图
- [ ] 段落分隔清晰
- [ ] 重点内容有标识

### 平台适配检查

- [ ] 符合目标平台的内容规范
- [ ] 标签和分类选择合适
- [ ] 发布时间选择合适

## 平台规则提醒

### 微信公众号

| 违规类型 | 后果 |
|----------|------|
| 诱导分享 | 限流 |
| 虚假信息 | 删文/封号 |
| 敏感内容 | 封号 |
| 侵权内容 | 投诉下架 |

### 知乎

| 违规类型 | 后果 |
|----------|------|
| 无根据攻击品牌 | 删回答/禁言 |
| 引战内容 | 删内容 |
| 违规引流 | 禁言 |

### 小红书

| 违规类型 | 后果 |
|----------|------|
| 虚假宣传 | 限流/删笔记 |
| 敏感词 | 限流 |
| 图片问题 | 限流 |
