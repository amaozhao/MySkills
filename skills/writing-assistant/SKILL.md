# Writing Assistant

## 概述

全流程文章写作助手，从选题到复盘，支持公众号、知乎、小红书、博客等多平台。

## 核心理念（借鉴 GitHub 最佳实践）

| 来源 | 借鉴点 |
|------|---------|
| copywriting | 阶段化 + 硬门槛（Copy Brief Lock） |
| brainstorming | 决策锁（需用户确认才能进入下一阶段） |
| seo-content-writer | E-E-A-T 信任要素 + 输出规范 |

## 工作流程

```
选题 → 选题确认（硬门槛）→ 大纲 → 大纲确认（硬门槛）→ 写作 → 标题 → 发布 → 复盘
```

> **重要**：每个阶段有"硬门槛"，必须用户确认后才能进入下一阶段

## 模块说明

### 1. 选题助手
- 自动搜索热门话题（可选）
- **选题确认（硬门槛）**：用户必须确认选题后才能进入大纲阶段
- 选题决策 5 问流程
- 评估选题价值

### 2. 大纲助手
- 4 种文章类型模板
- **大纲确认（硬门槛）**：用户必须确认大纲后才能进入写作阶段
- 按平台调整结构

### 3. 写作助手
- E-E-A-T 信任要素
- 开头/正文/结尾写法
- 速度提升技巧
- 常见问题解决

### 4. 标题助手
- 标题要素
- 生成技巧（3-5个备选）
- **输出规范**：带标题备选和推荐理由
- 评估标准

### 5. 发布助手
- 平台配置
- 发布时间
- **输出规范**：完整检查清单

### 6. 复盘助手
- 数据指标解读
- **输出规范**：标准复盘报告格式
- 改进方向
