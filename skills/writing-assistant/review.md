# 复盘助手

## 概述

帮助你分析发布后的数据，总结经验，给出改进建议。

## 工作流程

```
发布完成 → 等待48小时 → 收集数据 → 数据分析 → 经验总结 → 改进建议
```

## 输出规范

借鉴 seo-content-writer，每次复盘输出：

```
【数据复盘报告】

📊 核心数据
- 阅读量：[数字]（参考值：>500为良好）
- 点赞率：[数字]%（参考值：>3%为优秀）
- 分享率：[数字]%（参考值：>1%为优秀）
- 完读率：[数字]%（参考值：>50%为良好）

📈 数据诊断
- [组合分析结果]

📋 改进建议
1. [具体建议1]
2. [具体建议2]
3. [具体建议3]

📝 下次参考
- 适合方向：[]
- 不适合方向：[]
```

---

## 数据指标解读

### 核心指标

| 指标 | 含义 | 参考值 |
|------|------|--------|
| 阅读量/浏览量 | 内容曝光度 | >500 为良好 |
| 点赞率 | 内容认可度 | >3% 为优秀 |
| 分享率 | 传播力 | >1% 为优秀 |
| 完读率 | 阅读完成度 | >50% 为良好 |
| 评论数 | 用户反馈 | >10 为活跃 |

### 指标组合分析

| 组合 | 诊断 |
|------|------|
| 高阅读 + 低点赞 | 标题党，内容不够好 |
| 高点赞 + 低分享 | 有价值，但缺乏传播点 |
| 高分享 + 低评论 | 观点引发共鸣，但互动性弱 |
| 高完读 | 内容有价值，结构好 |

---

## 数据复盘维度

| 维度 | 分析要点 |
|------|----------|
| 选题 | 是否击中目标读者需求 |
| 标题 | 是否有足够的吸引力 |
| 开头 | 是否成功抓住读者 |
| 内容 | 是否有实际价值 |
| 结尾 | 是否有有效引导 |
| 时机 | 是否选对发布时间 |
| 配图 | 是否有视觉吸引力 |

---

## 改进方向

| 问题表现 | 改进策略 |
|----------|----------|
| 阅读量低 | 优化标题、改进选题 |
| 互动率低 | 增加干货、强化观点输出 |
| 分享率低 | 增强情感共鸣、结尾引导转发 |
| 完读率低 | 优化开头、增强内容吸引力 |
| 评论少 | 结尾提开放性问题 |

---

## 复盘输出示例

```
【数据评估】
阅读量：523（中等偏上，>500为良好）
互动率：一般（点赞率2.3%，分享率0.6%）
爆款潜力：有

【数据解读】
- 分享率偏低，说明标题吸引力不足
- 收藏率不错（0.8%），说明内容有价值

【改进建议】
1. 标题优化：增加情绪词或悬念
2. 开头优化：增强钩子
3. 结尾优化：增加转发引导

【下次参考】
- 本文适合方向：技术工具对比
- 本文不适合方向：纯情绪共鸣
```
