# 微信公众号写作助手

## 概述

本助手提供从选题到发布全流程的微信公众号写作支持，包含五个独立模块，可根据需求自由组合使用。

## 使用场景

- 不知道写什么，需要选题建议
- 确定选题后，需要大纲模板
- 写完文章，需要标题优化
- 文章完成，准备发布
- 发布后，需要数据分析

## 模块结构

### 模块一：选题助手

当你不确定写什么主题时使用。选题助手会分析当前热点、评估选题价值、给出具体选题建议。

### 模块二：大纲助手

当你确定选题后使用。大纲助手会根据文章类型（工具对比、情绪共鸣、热点解读）生成对应的大纲模板。

### 模块三：标题助手

当你准备写文章时使用。标题助手会生成多个备选标题，并给出评估建议。

### 模块四：发布助手

当你写完文章准备发布时使用。发布助手会给出发布时间建议、配图清单、发布前检查清单。

### 模块五：复盘助手

当你发布文章48小时后使用。复盘助手会分析数据、总结经验、给出改进建议。

## 使用方式

### 输入参数

```json
{
  "modules": ["选题助手", "大纲助手", "标题助手"],
  "topic_type": "热点解读",
  "time_budget": "2-3小时",
  "has_material": false,
  "deadline": "今天21:00"
}
```

### 参数说明

| 参数 | 必填 | 可选值 | 说明 |
|------|------|--------|------|
| modules | 是 | 选题助手、大纲助手、标题助手、发布助手、复盘助手 | 可多选，用逗号分隔 |
| topic_type | 否 | 工具对比、情绪共鸣、热点解读、日常感悟 | 选题类型 |
| time_budget | 否 | 2-3小时、3-4小时、5小时+ | 时间预算 |
| has_material | 否 | true、false | 是否有现成素材 |
| deadline | 否 | 今天、明天、本周 | 发布截止时间 |

### 快捷指令

| 指令 | 含义 |
|------|------|
| 写热点文 | 自动调用选题助手＋大纲助手＋标题助手 |
| 帮我起标题 | 只调用标题助手 |
| 准备发布 | 只调用发布助手 |
| 分析数据 | 只调用复盘助手 |
| 完整流程 | 调用全部五个模块 |

## 文章类型与写作模板

### 类型一：工具对比文

**适用场景**：比较两个或多个AI工具、产品的优劣。

**读者价值**：帮助读者做出选择决策。

**大纲模板**：

```
【标题】用了XX和XX后，我发现了X个惊人差异

【开头（150字以内）】
- 场景代入：什么场景下用的
- 冲突：有什么问题或发现
- 预告：本文会告诉你什么

【正文结构】
一、先说结论（300字）
- 2个工具的定位
- 我的推荐（给不同人群）

二、价格对比（500字）
- 官方价格
- 实际使用成本
- 哪个更划算

三、使用体验对比（800字）
- 速度对比
- 质量对比
- 稳定性对比
- 每个点用实际案例说明

四、适用场景（500字）
- 什么情况用A
- 什么情况用B
- 什么情况都用

五、最终建议（300字）
- 明确推荐
- 给出使用建议

【结尾】
- 总结
- 引导互动
```

**字数参考**：2000-3000字

### 类型二：情绪共鸣文

**适用场景**：讲述个人经历，引发读者情感共鸣。

**读者价值**：获得情感支持、行动启发。

**大纲模板**：

```
【标题】42岁被裁那天，我在XX坐了X小时

【开头（200字以内）】
- 场景：具体的时间、地点、动作
- 情绪：那一刻的感觉
- 转折：我决定...

【正文结构】
一、那个瞬间（400字）
- 详细描述发生了什么
- 细节越多越好（让人代入）
- 不是抱怨，是真实

二、为什么难（500字）
- 不是钱的问题，是...
- 不是丢脸，是...
- 点出深层痛点

三、我做了什么（600字）
- 经历了哪些挣扎
- 如何走出来的
- 关键转折点

四、给同样处境的你（500字）
- 不是鸡汤，是建议
- 具体可操作
- 真实不夸大

【结尾】
- 一句话总结
- 引导互动
```

**字数参考**：1800-2500字

### 类型三：热点解读文

**适用场景**：解读当前热点事件，提供独特视角。

**读者价值**：了解事件背后的逻辑、获得独特观点。

**大纲模板**：

```
【标题】XX刷屏了，我来泼点冷水/说点真话

【开头（150字以内）】
- 简单交代发生了什么
- 你的独特立场

【正文结构】
一、发生了什么（300字）
- 客观描述事件
- 关键时间线

二、为什么刷屏（400字）
- 分析传播原因
- 站在不同角度看

三、我的专业解读（800字）
- 你的独特角度
- 技术/商业/社会层面分析
- 不是复述新闻，是分析

四、对普通人的影响（500字）
- 利好哪些人
- 利空哪些人
- 需要注意什么

五、我的判断（300字）
- 预测未来
- 给出建议

【结尾】
- 观点总结
- 引导讨论
```

**字数参考**：2000-3000字

### 类型四：日常感悟文

**适用场景**：分享日常生活感悟、心得。

**读者价值**：获得启发、情感共鸣。

**大纲模板**：

```
【标题】重启第X天：关于XX的一点思考

【开头（150字以内）】
- 引入话题
- 抛出观点

【正文结构】
一、背景交代（300字）
- 发生了什么
- 我的处境

二、核心观点（600字）
- 我的思考
- 我的行动

三、经验总结（400字）
- 我学到了什么
- 对你有何启发

【结尾】
- 升华
- 引导互动
```

**字数参考**：1500-2000字

## 发布策略

### 最佳发布时间

| 时间段 | 特点 | 适用文章类型 |
|--------|------|--------------|
| 早07:00-08:30 | 通勤场景，碎片刷手机 | 短内容、快阅读 |
| 午12:00-13:30 | 午休时间，躺床上刷 | 中等内容 |
| 晚20:00-22:00 | 黄金时间，注意力集中 | 深度内容 |

### 推荐发布时间

| 场景 | 推荐时间 | 理由 |
|------|----------|------|
| 深度长文 | 今晚21:00 | 用户有耐心读完 |
| 追热点 | 越快越好 | 时效性优先 |
| 日常感悟 | 今晚21:00 | 互动率高 |

### 发布前检查清单

**内容检查**：
- [ ] 开头100字有钩子
- [ ] 有至少1个真实故事
- [ ] 有至少3个实用建议
- [ ] 结尾有总结+引导

**标题检查**：
- [ ] 标题能引起好奇
- [ ] 标题有情绪
- [ ] 标题承诺了价值

**排版检查**：
- [ ] 有封面图
- [ ] 有3-5张配图
- [ ] 段落长度合适（3-4行一段）
- [ ] 有小标题分隔

**发布检查**：
- [ ] 封面图准备好
- [ ] 摘要写好
- [ ] 发布时间选好

## 数据分析

### 数据指标解读

| 指标 | 含义 | 理想值 |
|------|------|--------|
| 阅读量 | 文章曝光度 | >500为良好 |
| 点赞率 | 内容认可度 | >3%为优秀 |
| 分享率 | 传播力 | >1%为优秀 |
| 收藏率 | 实用价值 | >0.5%为良好 |
| 评论数 | 互动度 | >10为活跃 |

### 数据复盘维度

| 维度 | 分析要点 |
|------|----------|
| 选题 | 是否击中读者需求 |
| 标题 | 是否有吸引力 |
| 开头 | 是否抓住读者 |
| 内容 | 是否有价值 |
| 结尾 | 是否有引导 |
| 时机 | 是否选对时间 |
| 配图 | 是否有视觉吸引 |

### 改进方向

| 问题 | 改进策略 |
|------|----------|
| 阅读量低 | 优化标题、增加追热点 |
| 点赞率低 | 增加干货、强化观点 |
| 分享率低 | 增强情绪、结尾引导 |
| 收藏率低 | 增加实用建议 |
| 评论少 | 结尾提开放性问题 |

## 常见问题

### 问题一：不知道写什么怎么办？

**答案**：从生活中找素材。从你的真实经历、遇到的问题、学到的东西入手。真实的故事最有共鸣。

### 问题二：追热点来不及写怎么办？

**答案**：如果超过48小时，就不要追了。可以换一个角度，写深度分析或后续跟进。

### 问题三：阅读量一直很低怎么办？

**答案**：检查标题是否有吸引力，检查开头是否抓住人，检查内容是否有干货。也可以参考同类高阅读量文章。

### 问题四：写不出来怎么办？

**答案**：先写大纲，大纲是骨架，先把骨架搭起来，再填肉。

### 问题五：数据焦虑怎么办？

**答案**：发完48小时后再看，前24小时不要频繁刷新后台。把精力放在下一篇文章上。

## 注意事项

本助手能做的：
- 选题分析和建议
- 大纲模板生成
- 标题建议
- 发布策略建议
- 数据分析和复盘

本助手不能做的：
- 直接生成完整文章（需要你自己写）
- 配图生成（只能给提示词）
- 代替你做决策（只给建议）
- 保证阅读量（取决于内容和执行）
